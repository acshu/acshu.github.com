Transit Analytical Curve maker - Help

The TAC-maker screen has 5 sections:

  • Input parameters - Sets parameters of the system
  • Import settings - Imports observation and transforms observation to phases/fluxes
  • Plot - Displays observation points and synthetic curve in a plot with corresponding residuals
  • Result table - Displays result of the calculations along with a export function
  • File menu - Saves and opens model file
Tips that are useful when working with TAC-maker

Installation:

Unzip the archive and run TAC-maker.exe file inside TAC-maker directory.

Input parameters

  • Semi-major axis - Radius of the orbit of transiting planet in AU
  • Stellar radius - Radius of the star in AU or Rjup(conversion AU <-> Rjup is done automatically)
  • Planet radius - Radius of the planet in AU or Rjup(conversion AU <-> Rjup is done automatically)
  • Stellar temperature - Temperature of the star in Kelvin scale
  • Inclination - Inclination of the orbit from the perspective of the observer in degrees
  • Darkening law - The stellar surface darkening law used for calculations
  • Darkening coefficient 1 - First coefficient for darkening law calculation
  • Darkening coefficient 2 - Second coefficient for darkening law calculation
  • Phase end - The end phase of the calculation typically < 0.05
  • Phase step - Phase end / Phase step = total points which will be calculated
  • Integration precision - How small the error of integration have to be

Import settings

  • Browse... button - Loads a tab separated file in (phase or HJD) (flux or magnitude) format
  • Convert HJD to phases - Check this option if your import file is in HJD
    • T0 - HJD time of the transit
    • P - Orbital period of the planet in HJD

  • Convert magnitude to flux - Check this option if you file is in magnitudes
    • Mag - Max. magnitude of the system out of transit

  • Redraw button - Draws imported file on the plot

Plot

The "Plot" tab contains two figures, depending on whether a import file is loaded:

  • Top figure plots synthetic curve as blue line and imported points as red dots.
  • Bottom figure show the residual for synthetic and imported values along with a Î§2.

Result table

The "Result table" tab shows last calculation values. For each phase values of the synthetic and imported curve are show along with the difference between them. This table can be exported in tab separated file from the "Export..." button.

File menu

File menu is used for opening and saving your models.

You can use "data/" folder for storing your observations and models files. There is one example at "data/wasp10/":

  • curve.dat - Represents observation file
  • wasp10.ini - Represents already saved model which can be loaded from "File -> Open..." menu.

Tips

  • Keep "Integration precision" lower until you find rough estimate of the system this will reduce computation time.
  • Set "Phase step" to 1 if you have a lot of observation points in this case synthetic curve will be generated only for imported phases and this will reduce the time for computation.
  • Your last session is automatically save on exit and next time you start the program it will load "Input parameters" and "Import settings" sections.
